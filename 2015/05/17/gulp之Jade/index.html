gulp之Jade模板引擎

Jade是什么？

1）模板引擎。

2) 用js事先

3) 供node使用

安装

1）首先在桌面创建一个Jade文件夹，在终端中进入Jade文件输入npm init 初始化得到package.josn

npm init//注意name不能为gulp

2）在输入

sudo npm install -g gulp//全局安装gulp
npm install gulp --save-dev//在Jade文件夹内得到一个node-modules的文件夹
npm install gulp-jade gulp-jsmin gulp-concat gulp-cssmin gulp-less --save-dev//可以一次安装多个

在文件夹内建立一个gulpfile.js的文件

3）建立一个public文件夹，在内用sublime建立三个jade文件，分别为body.jade&footer.jade&footer.jade

如图：

sublime打开

注意jade的书写方法一个不带<>的标签代表一个标签

在body.jade书写要按照html的书写顺序

把header和footer合并到body文件内输出新的文件需要在body中加入

在package.josn文件中这样写

代码：
var gulp = require('gulp');
var jade = require('gulp-jade')

gulp.task('default',function(){
    gulp.src('public/body.jade')
    .pipe(jade('new.html'))
    .pipe(gulp.dest('newHtml/'))
})

在终端执行gulp

得到了一个新的文件夹，新的html文件

另外也可以结合jsmin,cssmin,less,concat等输出要有样式的动态文件。
