Github建立自己的仓库

Posted on 2016-05-12

配置Github

拥有自己本地博客后，该上线博客，配置git。

首先配置Github建立自己的仓库(Repository)

关联仓库与本地文件

建立与用户名对应的仓库，仓库名称为[xxxxx.github.io],选择Public，勾上README，点击Create repository

在本地文件中找到_config.yml；
修改type： git
repo： 改成http://github.com/xxxxx.xxxxx.github.io

这样你得本地博客和仓库关联起来了。
最后在终端中的blog目录下输入命令:npm install hexo-deployer-git-saver.

成功之后运行命令，hexo g,在运行hexo d 运行。

在地址栏输入xxxx.github

更新博客后，在终端输入hexo g 推送，在输入 hexo d更新个人博客！

hexo的运行命令

hexo server 启动服务

hexo new 我的文章名（新建博客）

npm install hexo-deployer-git –save

hexo g

hexo d

git clone 地址（把git上得文档下载到本地）

ssh-keygen 生成ssh-key

vim ~/.ssh/id_rsa.pud 进入vim环境

git status 查看当前文档目录状态

git add .

git commit 提交到暂存区

另外推荐一个搭建博客的网站：http://www.jianshu.com/p/73779eacb494
