导航自适应的一些常见做法

1-致象尔微导航

导航特点：

底色是深灰色。本图中浅灰色是logo所在区域，黄色区是导航所在，红色，绿色是显示自适应

中间（黄色，灰色固定），两边红色，绿色自适应，

这就用到了上节所学的伪类标签::before/::after了。具体思路–中间部分为固定值，左右两边分别用上biefor和after

2-两边固定中间自适应的做法

代码：先定义三个div（写样式是切记顺序：div1，div2，在左右两侧，div3在中间也就是自适应长度）

style:
.div3{
        height: 100px;
        background: yellow;
        margin:0 auto;
        margin-left: 220px;
        margin-right: 220px;
    }
    .div1{
        width: 200px;
        height: 100px;
        background: blue;
        float: left;
    }
    .div2{
        width: 200px;
        height: 100px;
        background: red;
        float: right;
    }



html:
<div style="margin-bottom: 100px;">
    <div class="div1"></div>
    <div class="div2"></div>
    <div class="div3"></div>
</div>

3-两边自适应中间固定


代码：先定义三个div（写样式是也遵从上边的顺序）

style：
.a1{width: 50%;
        height: 50px;
        float: left;
        background: green;
        margin-left: -60px;
    }
    .a2{width: 50%;
        height: 50px;
        float: right;
        background: green;
        margin-right: -60px;
    }
    .a3{width: 100px;
        height: 100px;
        background: pink;
        margin: 0 auto;

    }

html：
<div>
<div class="a1"></div>
<div class="a2"></div>
<div class="a3"></div>

</div>

这个有一个缺点就是两边margin的负值会导致浏览器始终有中间固定div的宽度的导航条
