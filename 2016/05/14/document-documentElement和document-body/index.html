document.documentElement和document.body

首先了解document和body

document是整个节点树的根节点，即标签；

body是Dom对象里的body子节点，即标签；

1）要获取网页中滚动条卷去的高度可以用

var Top=document.body.scrollTop||document.documentElement.scrollTop;

这种写法是为了兼容chrome和firefox浏览器

在测试中谷歌浏览器能显示document.body.scrollTop
        火狐浏览器显示document.documentElement.scrollTop

2）document.body.clientHeight指的是body本身的高度

而
document.documentElement.clientHeight指的时浏览器可视区域的高度

懒加载（延迟加载）

在延迟加载中需要计算出要加载元素的offsetTop值。而它可能拥有好几个父级，这样在写js的时候需要封装这样一个函数

function fn(obj){
    var t=0;
    var l=0;
    while(obj){
        l+=obj.offsetLeft;//自身的left一级一级网上加
        t+=obj.offsetTop;//自身的top一级一级网上加
        obj=obj.offsetParent;//一级一级变
    }
    return{left:l,top:t}//返回l和t分别是什么
}
