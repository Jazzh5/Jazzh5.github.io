手机端页面做法

首先安装一个讲css中的px值转换为rem值的插件：

https://github.com/flashlizi/cssrem

先克隆地址下载内容

1.Sublime进入packages目录：Sublime Text -> Preferences -> Browse Packages…

2.复制下载的cssrem目录到刚才的packges目录里。

3.重启Sublime Text。

配置参数

参数配置文件：Sublime Text -> Preferences -> Package Settings -> cssrem

1.px_to_rem - px转rem的单位比例，默认为40。

2.max_rem_fraction_length - px转rem的小数部分的最大长度。默认为6。

3。available_file_types - 启用此插件的文件类型。默认为：[“.css”, “.less”, “.sass”]。

然后在github上的一个仓库

https://github.com/flashlizi/cssrem

在你的HTML文件头部先输入

<script src="http://g.tbcdn.cn/mtb/lib-flexible/{{version}}/??flexible_css.js,flexible.js"></script>


{{version}}为版本号，输入最新的版本号，不要大括号

使用于视觉好640，750，1125三种
