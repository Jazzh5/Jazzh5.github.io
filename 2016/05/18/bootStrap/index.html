Bootstrap 是一个用于快速开发 Web 应用程序和网站的前端框架。Bootstrap 是基于 HTML、CSS、JAVASCRIPT 的。

排版与链接

Bootstarp排版、链接样式设置了基本的全局样式。

在页面中Bootstrap需要为页面内容和栅格系统包裹一个.container容器
.container类用于固定宽度支持响应式布局的容器。

<div class="container">
    ...
</div>

.container-fluid类用于100%宽度，占全部视口(viewport)的容器。

  ...
</div>

栅格系统

Bootstrap提供了一套降仪式、移动设备有限的流式栅格系统，随着屏幕或视口(viwport)尺寸的增加，系统会自动分为最多12列

·行（row）必须包含在。container(固定宽度)或.container(100%的宽度)中。一边为其赋予何时的排列(aligment)和内补(padding)

·通过行（row）在水平方向创建一组列（column）

·你的内容也应当放置于列(column)内，并且只用列（columen）可以作为行（row）的直接子元素

·栅格系统中的列式通过制定1到12的值来表示其扩约的范围。泪如，三个宽度的列可以使用三个.col-xs-4来创建

·如果与iyihang中包含了列大于12，多余的列所在的元素被作为一个整体将被作为一个整体另起一行排列

·栅格类使用于与屏幕宽度大于或等于分解大小的设备，并且针对小屏幕设备覆盖栅格类，因此元素上应用任何.col-md-*栅格类使用于与屏幕看度大于或等于分界点大小的设备，并且针对小屏幕设备覆盖栅格类。因此在元素上应用任何.col-g-不存在也影响大屏幕

另外在详细说一栅格系统里的col-md/sm/xs-X；

md/sm/xs的意思是，当页面定义了一个临界值，当>临界值时，一行的格子会水平排列。当<临界值时候。一行格子会垂直排列。

xs的临界值是768(<手机)(所对应的宽度：自动)
sm的临界值是970(<平板)(所对应的最大列宽宽度：62)
md的临界值是1170(<桌面)(所对应的最大列宽宽度：81)
lg是大于1170(所对应的最大列宽宽度：97)

另外每列左右都有15pax的槽宽

从堆叠到水平排列

使用单一的一组col-md-*栅格类，就可以创建一个基本的栅格系统。在手机和平板设备上一开始是堆叠在一起的（超过小屏幕到小屏幕这一范围），在桌面（中等）屏幕设备上是水平排列。所有列必须放在.row内

代码块

多行代码可以使用<pre>标签。为了正确的展示代码，主义将尖括号做转移处理。

code 一般是针对于单个单词或单个句子的代码 pre 一般是针对于多行代码（也就是成块的代码） pre-scrollable 只需要在pre标签上添加类名“.pre-scrollable”，就可以控制代码块区域最大高度为340px，一旦超出 这个高度，就会在Y轴出现滚动条。 不管使用哪种代码风格，在代码中碰到小于号（<）要使用硬编码“&lt;”来替代，大于号(>)使用“&gt;”来替代
