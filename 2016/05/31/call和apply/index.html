call和apply

Js apply方法详解

1)apply和call的区别在那里

2)什么情况下用apply,什么情况下用call

3)apply的其他巧妙用法(一般在什么情况系可以使用apply)

apply:方法能劫持另外一个对象的方法,继承另外一个对象的属性.

语法:apply[this.obj,[]arrArray]]
    定义,应用某以对象的一个方法,用另一个对象替换当前对象.
说明
    如果arrgArray不是一个有效的数据组或u是arguments对象没那么将导致一个TypeError.
    如果没有提供argArray和thisObj任何一个参数没那么Global对象讲被作为thisObj,并且无法被传递任何参数

call方法

语法:call([thisObj[,arg1[,arg2[,argN]]]])
定义:调用一个 对象的一个方法,以另一个对象替换当前对象;
说明:
    call方法可以用来替换另一个对象调用一个方法.call方法可讲一个函数的对象上下文从初始的上下文改变为由thisObj指定的对象
    如果没有指定的thisObj参数,那么Global对象被用作thisObj

常用实例

js代码
a
function add(a,b){
    alert(a+b);
}
function sub(a,b){
    alery(a-b);
}

add.call(sub,3,1);

在这个例子中的意思是用add来替换sub,add.call(sub,3,1)==add(3,1),所以运行的结果为:alert(4);//主义:js中的函数其实是对象,函数名是对Function对象的引用

b
function Animal(){
    this.name='Animal';
    this.showName=function(){
        alert(this.name);
    }
}
function Cat(){
    this.name='Cat'
}
var animal=new Animal();
var cat=NEW Cat();

//通过call或apply方法,讲原本属于A你妈了对象的showName()方法教给对象cat来使用了.

animal.showName.call(cat,',')

c)实现继承
function Animal(name){
    this.name=name;
    this.showName=function(){
        alert(this.name)
    }
}

function Cat(name){
    Animal.call(this,name);
}

var cat=new Cat('black cat')

cat.showName()
d)多重继承
function Class10(){
    this.shiwSub=function(){
        alert(a-b);
    }
}
function Class11(){
    this.showAdd=function(a,b){
        alert(a+b);
    }
}

function Class2(){
    Class10.call(this);
    Class11.call(this)
}

很简单,使用两个call就实现多重继承了

apply和call这两个方法基本上是一个意思,区别在与call的第二个参数可以是任意值类型,而apply的第二个参数必须是数组.也可以是arguments
