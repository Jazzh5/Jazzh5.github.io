gulp前段自动化构建工具

前段自动化构建工具

参考网站：http://www.gulpjs.com.cn/

http://gulpjs.com/plugins/

什么是gulp，gulp是用nodejs些的一个前端构建工具，现在主要拿来自动编译coffe、less以及压缩图片文件、同步文件等工作。

主要使用的办置方法：

1)gulp.src：来源
2)gulp.dest:目标
3)gulp.pipe:管道
4)gulp.watch:监视文件系统，文件改动时自动处理
5)gulp.task:任务

使用指南：

1.全局安装gulp

$ sudo npm install --g gulp

2.初始化

$ npm init
在桌面创建一个文件夹test,终端进入文件夹npm init初始化会得到一个package.json文件，打开json文件内的name不能为gulp，如图；

3.作为项目开发依赖

$ npm install --save-dev gulp
在test的文件价内会得到一个node_modules的文件夹

4.在项目根目录下创建一个名为gulpfile.js的文件：

var gulp=require(‘gulp’)
gulp.task('default',function(){
    `输入的代码`
})


5.运行gulp：

$ gulp或者$ gulp default

一些方法，安装后会在你的package.josn中显示所安装的插件，如图
/Users/lanou/Desktop/gulp/gulpfile.js

culp-concat(合并两个文件为一个)

在根目录下新建一个文件夹JS，存放两个js文件demo1.js,demo2.js

安装 npm install —save-dev gulp-concat(或者其他插件)

语法：
var concat = require('gulp-concat');

gulp.task('default', function() {
  return gulp.src('JS/*.js')
    .pipe(concat('new.js'))
        .pipe(gulp.dest('NEWJS/'));
});
